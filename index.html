Nadameu

Coleção de scripts de usuário para o complemento Greasemonkey, do Firefox.

Instalação

É necessário utilizar o Mozilla Firefox com o complemento Greasemonkey instalado.

Para instalar o complemento Greasemonkey, clique aqui e selecione a opção “Add to Firefox”. Se necessário, reinicie o Firefox após a instalação.

Com o complemento instalado, selecione um script abaixo para instalá-lo.

Após instalados, os scripts serão atualizados automaticamente sempre que houver uma versão nova.

Scripts

  • RENAJUD: Facilita a utilização do sistema RENAJUD por usuários da Justiça Federal da 4ª Região.
  • Ponto eletrônico: Facilita a utilização do ponto eletrônico da JFPR.
  • SEI!: Facilita a utilização do sistema SEI! do TRF4.
  • Scripts para o e-Proc da Justiça Federal da 4ª Região:
    • Barra flutuante;
    • Sequência de localizadores: Altera a exibição da página do histórico de localizadores.
    • Impressão da tela de videoconferência agendada;
