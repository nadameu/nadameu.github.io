# Política de privacidade

Os scripts deste _site_ não armazenam informações sobre os usuários.

Em caso de necessidade de salvamento de dados "em nuvem", será solicitada
autorização para acesso ao serviço Google Drive particular do usuário.

Os dados acessados através do Google Drive estão disponíveis apenas para o
usuário e não são armazenados em outro local ou compartilhados, nem mesmo com o
desenvolvedor dos scripts.

# Termos de serviço

Você pode usar os scripts deste _site_ instalando-os em seu computador.

Você pode alterá-los para uso pessoal.

Se você compartilhar os scripts deste _site_ com outras pessoas, alterados ou
não, deverá informar a tais pessoas o endereço deste _site_ para que possam
verificar a existência de versões mais recentes ou conferir as alterações
realizadas.
